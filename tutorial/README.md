# GenSec tutorial example

This folder contains the MePTCDI-on-graphene tutorial used by
`tutorial.ipynb`.

Main contents:

- `tutorial.ipynb`: explanatory notebook for the full workflow.
- `setup/`: starting files for generation, FPS, and training.
- `atoms/`: isolated-atom reference-energy setup for MACE `E0s`.
- `final/`: completed example run used for inspection in the notebook.
- `search/`: clean search-only run using the trained model.
- `assets/`: PNG images used by the notebook.
- `requirements-notebook.txt`: Python packages needed to run the notebook.

The notebook is meant to explain the workflow and input choices. The folders
hold the files to inspect or adapt when running the same stages outside the
notebook.
