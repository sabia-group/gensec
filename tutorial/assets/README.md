# Notebook assets

This folder contains images used by `tutorial.ipynb`.

- `tutorial-structures.png`: molecule and graphene-frame overview.
- `tutorial-generated-side-view.png`: side view of one accepted generated
  structure.
- `gui-overview.png`: parameter GUI overview screenshot.
- `gui-training.png`: parameter GUI training-page screenshot.

These files are generated or referenced by the notebook and can be replaced if
the example system or GUI changes.
