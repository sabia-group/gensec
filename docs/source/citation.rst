Citing GenSec
=============

Please cite GenSec using the thesis reference below.

BibTeX
------

.. code-block:: bibtex

	 @phdthesis{Maksimov-thesis,
		 author  = {Maksimov, Dmitrii},
		 title   = {Characterization and Prediction of Peptide Structures on Inorganic Surfaces},
		 school  = {EPFL},
		 address = {Lausanne},
		 year    = {2022},
		 doi     = {10.5075/epfl-thesis-9740},
		 url     = {https://infoscience.epfl.ch/handle/20.500.14299/189434}
	 }

If you use the code, please also cite the corresponding release DOI from Zenodo.
